# MAiD Infographic: The Ethics of Medical Assistance in Dying in Canada

A comprehensive, interactive digital infographic exploring the ethical complexities of Medical Assistance in Dying (MAiD) in Canada through the narrative journey of patient Alex.

## What's Included

This is a **fully static, production-ready build** with no dependencies or build tools required.

### Core Files
- **index.html** - Complete infographic (360 KB minified)
- **assets/** - CSS and JavaScript bundles
- **.nojekyll** - GitHub Pages configuration
- **DEPLOYMENT.md** - Deployment instructions

## Features

### Interactive Elements
- 🎙️ **Guided Tour** - Click "Play Full Narration" for a warm, natural voice narration of all sections
- 🔊 **Individual Section Audio** - Click speaker buttons (🔊) next to key sections for narration
- 📍 **Navigation Bar** - 9 sticky section links with active highlighting
- 📊 **Progress Bar** - Visual indicator of scroll completion
- ⬆️ **Back-to-Top Button** - Floating button for easy navigation
- 💭 **Hover Tooltips** - Definitions for key terms (autonomy, dignity, sanctity of life)
- 🤔 **Interactive Reflection** - "What Would You Do?" section with ethical choice buttons
- 🧠 **Thought Moments** - Alex's internal reflections throughout the narrative

### Content
- 14 comprehensive panels covering MAiD ethics
- Academic citations and references (APA format)
- Narrative-driven storytelling following patient Alex
- Arguments for and against MAiD
- Canadian legal history and timeline
- Ethical framework analysis
- Interactive decision moment
